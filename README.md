The Student Performance Tracker is a web-based application designed for teachers to manage and track student performance across multiple subjects. It allows you to:
Add students
Assign grades for subjects
View student details
Calculate average grades
The application is accessible online at:

https://student-performance-tracker-juck.onrender.com

1. Home Page
When you open the application, you will see the home page with the following options centered on the screen:
Add Student – Add new students to the system.
Add Grades – Assign grades for subjects to students.
View Student Details – View a student's grades and information.
Calculate Average – Calculate a student’s average grade across all subjects.

3. Add Student
Click “Add Student” on the home page.
Fill in the following details:
Name – Enter the student’s full name.
Roll Number – Enter a unique roll number for the student.
Click “Add Student”.
A message will confirm whether the student was added successfully or if the roll number already exists.
Example:
Name: A
Roll Number: 101

3. Add Grades
Click “Add Grades” on the home page.
Fill in the following details:
Roll Number – Enter the student’s roll number.
Subject – Enter the subject name (e.g., Math, Science).
Grade – Enter a number between 0 and 100.
Click “Add Grade”.
A confirmation message will show if the grade was successfully added.
Example:
Roll Number: 101
Subject: Math
Grade: 95

4. View Student Details
Click “View Student Details” on the home page.
Enter the Roll Number of the student you want to check.
Click “View Details”.
The page will display:
Student Name
Roll Number
Grades in all subjects
Example Output:
Name: A
Roll Number: 101
Grades:
Math: 95
Science: 88

5. Calculate Average
Click “Calculate Average” on the home page.
Enter the Roll Number of the student.
Click “Calculate Average”.
The page will display:
Student Name
Roll Number
Average Grade (rounded to 2 decimal places)

Example Output:
Name: A
Roll Number: 101
Average Grade: 91.50
